# 심리학 채널 풀 셋업 (통합판)

> **추천 AI**: Claude 또는 ChatGPT  
> 💡 버전 A(기본)와 버전 B(상세)를 단계별로 선택해 쓸 수 있게 통합

## 프롬프트

```
1. 채널명 생성
(채널 링크 붙여넣기) < 이 채널을 분석해서 나만의 독특한 채널명 10개 추천해줘

2. 캐릭터 생성
(채널 캐릭터 캡처 업로드) < 업로드한 캐릭터를 참조해서 나만의 독특한 캐릭터를 만들거야.
간단한 애니메이션 스타일 캐릭터. 이미지 생성하지 말고 이미지 프롬프트만 만들어줘.
+ 채널 로고 이미지 프롬프트도 같이 만들어줘.

3. 영상 아이디어 10개
분석한 채널을 토대로 나만의 영상 아이디어 10개를 생성해줘

4. 대본 생성
(선택한 영상 아이디어 붙여넣기) < 공백 제외 3,000자로 대본 작성해줘.
인트로는 시청자 이탈을 막기 위해 후킹이 포함돼야 해.

5. 이미지 프롬프트
(예시 이미지 캡처본 첨부) < 이 이미지를 참고해서
각 문장마다 이미지 프롬프트 만들어줘. 이미지 생성 하지마.

6. 썸네일
(예시 썸네일 첨부) < 이를 참고해서
(영상 주제)에 대한 썸네일 이미지 프롬프트 만들어줘.
```
