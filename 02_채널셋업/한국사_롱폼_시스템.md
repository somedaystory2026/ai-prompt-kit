# 한국사 롱폼 4단계 시스템

> **추천 AI**: Claude  
> ⚠️ AI가 생성한 실록 인용구는 반드시 [국사편찬위원회](https://sillok.history.go.kr)에서 팩트체크 할 것

## 1단계 — 롱폼 최적화 주제 20선 추출

```
너는 100만 구독자를 가진 역사 유튜브 채널의 메인 작가야.
조선왕조실록에 기록된 100% 실제 역사적 사실 중에서,
10~15분 분량의 유튜브 롱폼 영상으로 만들기 좋은 흥미로운 주제 20가지를 뽑아줘.
단순히 왕의 일대기가 아니라 범죄 수사, 기이한 미스터리, 궁중 스캔들, 권력의 암투 등
시청자의 도파민을 자극할 수 있는 구체적인 사건 위주로 선정해 줘.
각 주제별로 1~2줄의 짧은 흥미 포인트를 함께 적어줘.
```

## 2단계 — 기승전결 줄거리 요약

```
[주제 번호와 제목을 이곳에 넣어주세요]으로 영상을 제작할게.
이 사건의 전체적인 흐름을 기(도입)-승(전개)-전(위기 및 절정)-결(결말 및 여운) 4단계로 나누어 요약해 줘.
모든 등장인물부터 먼저 소개해주고, 인물들의 동기와 시대적 배경이 잘 드러나도록 정리해 줘.
```

## 3단계 — 실록 인용 기반의 구어체 완벽 대본 생성 (4000~5000자)

```
방금 요약해 준 기승전결 줄거리를 바탕으로 유튜브 롱폼 대본을 작성해 줘.
시각/청각 효과 지시문은 모두 빼고, 오직 나레이션과 대사로만 구성해 줘.

오프닝 훅 공식 (3가지 중 1개 선택):
A. 클라이맥스 먼저 던지기: 사건의 가장 긴장되는 순간으로 냅다 시작
B. 완벽한 모순: 성군의 이미지 vs 실록의 기괴한 실제 행동을 극단적으로 대비
C. 충격적인 결말 스포일러: 비참한 최후를 먼저 던지고 "왜 이런 꼴이 됐는지" 유도

필수 포함 — 실록 원문 직접 인용:
"실제 조선왕조실록의 기록을 보면, 이 끔찍한 상황을 이렇게 건조하게 적고 있습니다. '...'"

구어체로 흥미있고 재미있게 (~했는데요, ~인 겁니다, 과연 어떻게 되었을까요?)
```

## 4단계 — 이미지 프롬프트 생성 (표 형식)

```
[3단계에서 만들어진 대본을 이곳에 붙여넣어 주세요]

공통 프롬프트 규칙:
화풍: 조선시대 진품 풍속화(Pungsok-hwa)와 산수화(Sansu-hwa)를 결합한 애니메이션 스타일.
거친 붓 터치, 전통 수묵담채 효과, 전통 안료 색감(청색, 갈색, 먹색, 차분한 적색)
품질: 8K 해상도 / 텍스트 금지 / 꽉 찬 배경 구도

출력 형식 (Markdown 표):
| 대본 내용 | 이미지 프롬프트 (영어) | 비디오 프롬프트 (영어) |
```
