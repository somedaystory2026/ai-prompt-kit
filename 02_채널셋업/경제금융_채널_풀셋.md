# 경제·금융 채널 1~6단계 풀셋

> **추천 AI**: ChatGPT 또는 Claude

## 1단계 — 채널명 최적화

```
내 채널은 투자, 재테크, 개인 금융을 전문으로 다뤄.
'___ 투자' (예: '샘 투자') 같은 형식으로 채널명 10개 추천해줘.
```

## 2단계 — 캐릭터 프롬프트

```
정확히 똑같은 스타일로 [캐릭터 특징 묘사]의 프로필 사진을 만들어줘.
참고 이미지랑 그림 스타일·비율·크기를 똑같이 맞춰야 해.
캐릭터는 정면을 보고 카메라를 똑바로 응시하게 해줘.
명암·그림자가 일관된 '패밀리 가이' 스타일 일러스트로 그려줘.
(캐릭터 스크린샷 첨부)
```

## 3단계 — 영상 아이디어 20개

```
금융·돈 관련 채널에 올릴 독창적인 영상 아이디어 20개를 만들어줘.
대중들이 좋아할 만하면서도, 흔하지 않고 잘 모르는 흥미로운 주제여야 해.
첨부한 스크린샷 채널과 비슷한 느낌으로 만들어줘.
(채널 영상 스크린샷 첨부)
```

## 4단계 — 대본 (닉 스타일)

```
이 제목으로 닉의 금융 교육 스타일을 살려서 유튜브 대본을 써줘:
'[여기에 영상 아이디어 붙여넣기]'

구조:
1. 오프닝 훅: '이거 내 얘긴데?'라고 생각할 공감 상황
2. 도입부: "내 이름은 닉이야. 난 [관련 금융 주제]에 대해서 진짜 미친 듯이 고민하거든.
   만약 네가 [시청자의 고민] 때문에 힘들다면, 구독 버튼 누르고 좋아요도 꼭 눌러줘."

스타일 가이드:
- 친구랑 말하듯이 자연스럽게 써
- 영상 초반에 사람들이 흔히 믿는 금융 상식을 깨부숴줘
- 실제 금액을 써서 수학적으로 딱딱 계산되는 예시를 보여줘
- 왜 그런 경제적 결정을 내리는지 심리적인 이유를 설명해줘
- 시청자가 반박할 만한 내용에 미리 대답해줘

분량: 10~15분 / 구독·좋아요 유도로 마무리
```

## 6단계 — 영상 이미지 프롬프트

```
너는 유튜브 금융 영상 전문 AI 비주얼 스토리보드 엔지니어야.
아래 대본을 줄별로 나눠서 각 줄마다 이미지 프롬프트 1개 + 영상 프롬프트 1개를 만들어줘.

이미지 프롬프트 필수:
- 똑같은 남자 캐릭터 전신 샷 (머리~발끝, 절대 잘림 없음)
- 워작 스타일의 금융 만화 / 흰색·밝은 배경
- 굵은 검은 테두리 / 평면적 색채 / 과장된 표정
- 이미지 안에 글자 넣지 마

영상 프롬프트 필수:
- 전신 캐릭터 미세하게 움직이기 (눈 깜빡임·숨쉬기·고개 까닥)
- 3~5초 / 천천히 줌인·패럴랙스 / 흔들림 없음

출력 형식:
대본 줄: (내용)
이미지 프롬프트: ...
영상 프롬프트: ...

[여기에 대본 붙여넣기]
```
