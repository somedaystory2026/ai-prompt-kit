# 모션 그래픽 Remotion 3종

> **추천 도구**: Claude Code + Remotion

## 1. 네온 삼각형 애니메이션

```
화면 중앙에서 네온 핑크색 삼각형이 회전하며 서서히 커지면서 나타나게 해줘.
그다음 삼각형의 세 꼭짓점에서 민트색 빛의 줄기 3개가 동시에 뿜어져 나와
삼각형 주위를 원형으로 빠르게 휘감아.
이 빛의 줄기들은 스피드 램프 효과를 주면서 회전 속도가 빨라졌다가 느려지기를 반복해.
마지막에는 삼각형이 반대로 회전하면서 모든 빛줄기를 안으로 끌어당기고,
아주 작은 점이 되어 순식간에 사라지는 연출을 해줘.
```

## 2. 글로우 공전 박스 (Remotion)

```
remotion 모범 사례랑 참고 이미지를 참고해서 알약 모양의 직사각형 박스를 하나 만들어 줘.
박스 주위에는 글로우(빛 번짐)가 궤도를 돌고 있어야 해 (레퍼런스 이미지처럼, 같은 색상 조합으로).
글로우들은 박스의 반대편에서 움직이고, 박스의 바깥 둘레를 따라 트래킹되어야 해.
부드러운 움직임으로, 빙글빙글 도는 식으로 말이야.
박스 안에는 텍스트가 없어야 해.
전체적으로 슈퍼 프리미엄한 느낌 - 인포그래픽이나 UI 모션 같은 분위기로.
```

## 3. 7개 박스 순차 등장 (Remotion)

```
remotion 모범 사례를 써서, 깔끔한 애플 감성의 직사각형 박스 7개 세트를 만들어줘.
각 박스는 애니메이션으로 등장해 - 블러 인, 페이드 인, 그리고 부드럽게 위로 슬라이드 업.
각 박스의 애니메이션은 2초에 걸쳐 진행되어야 해.

박스들은 다음 순서로 애니메이션돼야 해. 박스 안에 이 제목들을 넣어줘:
1. 효과 1 / 2. 효과 2 / 3. 효과 3 / 4. 효과 4 / 5. 효과 5 / 6. 효과 6 / 7. 보너스 효과
```
