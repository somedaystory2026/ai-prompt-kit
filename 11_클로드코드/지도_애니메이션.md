# 지도 애니메이션 (캘리포니아 → 뉴욕)

> **추천 도구**: Claude Code + Remotion

## 프롬프트

```
remotion 모범 사례를 사용해줘.
새 컴포지션을 만들고 지도를 추가한 다음,
캘리포니아에 포커스를 유지하면서 부드럽게 줌아웃해.
완료되면 캘리포니아에서 뉴욕까지 선을 애니메이션으로 그리고
카메라가 그 선을 따라가게 해줘.
전체적으로 부드럽고 키프레임 기반의 모션이어야 해.
이징 커브를 사용해서 가속과 감속이 자연스럽게 이루어지도록 해줘.
지도는 깔끔하고 프로페셔널해야 해.
갑작스러운 움직임이나 모션 글리치는 없어야 해.
```
