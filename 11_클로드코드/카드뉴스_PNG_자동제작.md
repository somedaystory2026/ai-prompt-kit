# 인스타그램 카드뉴스 PNG 자동 제작

> **추천 도구**: Claude Code  
> HTML/CSS로 카드뉴스를 만들고 Playwright로 PNG 이미지로 저장합니다.

## 프롬프트 (순서대로 사용)

```
1. Claude Code로 인스타그램 카드 뉴스를 만들고 싶은데 어떤 방법들이 있을까?
여러 방법 중에 HTML/CSS로 카드뉴스를 만들고 Playwright로 PNG 이미지로 저장하는 방법으로 하고 싶어.
이 방법으로 진행할 때 전체적인 흐름이 어떻게 되는지 단계별로 설명해줘.

2. 인스타그램 카드뉴스용 디자인 스타일을 3가지 추천해줘.
- 스타일 중 하나는 레드 계열 다크 톤 + 중앙 정렬 느낌으로
- 나머지 둘은 분위기가 서로 다른 스타일로 자유롭게 추천해줘
- 카드 비율: 4:5 / Python 기반으로 작업할 거야
- 3가지를 각각 별도 마크다운 파일로 저장해줘

3. 프로젝트 폴더에 있는 style-*.md 파일을 각각 참고해서
같은 내용의 샘플 카드뉴스를 3가지 스타일로 만들어줘.
- 카드 내용: 제목, 부제목, 태그, 하단 브랜드명
- 각 스타일당 5장씩 제작

4. 테스트 결과 [minimal/colorful/dark] 스타일로 진행하기로 했어.
주제: "[주제를 입력해주세요]"
이 주제로 카드뉴스를 만들어줘.
- style.md 스타일을 정확히 따를 것
- 핵심 포인트를 파악해서 5~10장 사이로 적절하게 구성
- 첫 장은 표지, 마지막 장은 마무리 + 팔로우 유도
- 하단 브랜드명: @[브랜드명]
- 출력: output/cards/card_01.png~ / output/cards/caption.md

5. 지금까지 한 모든 작업을 CLAUDE.md 파일에 정리해줘.
Claude Code의 새로운 세션에서 이 파일만 보고도 바로 작업할 수 있도록 작성해줘.
```
