# 인스타그램 카드뉴스 PNG 자동 제작

> **추천 도구**: Claude Code  
> HTML/CSS로 카드뉴스를 만들고 Playwright로 PNG 이미지로 저장합니다.

## 전체 흐름

```
1. 디자인 스타일 3종 선택 (마크다운 파일로 저장)
2. 샘플 카드뉴스 제작 (스타일당 5장)
3. 레이아웃 문제 자동 감지 및 수정
4. 최종 스타일 확정 후 실제 주제로 카드뉴스 제작
5. CLAUDE.md에 전체 작업 정리
```

---

## 프롬프트 (순서대로 사용)

### Step 1 — 방법 탐색 및 흐름 파악

```
Claude Code로 인스타그램 카드 뉴스를 만들고 싶은데 어떤 방법들이 있을까?
여러 방법 중에 HTML/CSS로 카드뉴스를 만들고 Playwright로 PNG 이미지로 저장하는 방법으로 하고 싶어.
이 방법으로 진행할 때 전체적인 흐름이 어떻게 되는지 단계별로 설명해줘.
```

### Step 2 — 디자인 스타일 3종 추천

```
인스타그램 카드뉴스용 디자인 스타일을 3가지 추천해줘.

아래 조건을 반영해줘:
- 스타일 중 하나는 레드 계열 다크 톤 + 중앙 정렬 느낌으로
- 나머지 둘은 분위기가 서로 다른 스타일로 자유롭게 추천해줘

각 스타일에 포함할 내용:
1. 색상 팔레트 (배경, 텍스트, 강조 색상)
2. 폰트 (제목, 본문)
3. 레이아웃 (정렬 방식, 구성 요소 배치)
4. 카드 구성 요소 (태그, 제목, 내용, 하단 브랜드 영역 등)
5. 카드 비율: 4:5
6. 분위기 키워드 3개

나중에 Claude Code가 이 파일을 디자인 기준으로 쓸 거야.
3가지를 각각 별도 마크다운 파일로 저장할 수 있게 내용과 파일명도 같이 작성해줘.
Python 기반으로 작업할 거야.
```

### Step 3 — 샘플 카드뉴스 3스타일 × 5장 제작

```
프로젝트 폴더에 있는 style-minimal.md, style-colorful.md, style-dark.md 파일을
각각 참고해서 같은 내용의 샘플 카드뉴스를 3가지 스타일로 만들어줘.

카드 내용 (3가지 스타일 모두 동일하게):
- 제목: 클로드를 써야 하는 이유
- 부제목: AI 챗봇, 이제 똑똑하게 골라 쓰자
- 태그: #AI #클로드
- 하단 브랜드명: @[브랜드명]

요구사항:
- 각 스타일 MD 파일의 색상, 폰트, 레이아웃을 정확히 따를 것
- 각 스타일당 5장씩 만들어줘
- 출력:
  - output/sample-minimal/card_01.png ~ card_05.png
  - output/sample-colorful/card_01.png ~ card_05.png
  - output/sample-dark/card_01.png ~ card_05.png
```

### Step 4 — 레이아웃 문제 자동 수정

```
output/sample-minimal/ 폴더의 카드들을 직접 확인해봐.
텍스트 겹침, 빈 공간, 여백 불균형 등 레이아웃 문제가 있으면
스스로 파악해서 전부 수정하고, 변경사항을 style-minimal.md에도 반영해줘.
확인했으면 아래 사항도 반영해줘.

- 제목을 좀 더 임팩트 있게 키워줘
- 배경 하단에 그라데이션 효과 추가해줘
- 태그 부분을 좀 더 눈에 띄게 스타일링해줘
```

### Step 5 — 파일 기반 카드뉴스 제작

```
테스트 결과 [minimal/colorful/dark] 스타일로 진행하기로 했어.

input 폴더 안에 있는 파일을 읽고 내용을 기반으로 카드뉴스를 만들어줘.

조건:
- style.md 스타일을 정확히 따를 것
- 내용의 핵심 포인트를 파악해서 5~10장 사이로 적절하게 구성해줘
- 각 카드는 한 가지 메시지만 담고, 텍스트는 핵심만 임팩트 있게
- 첫 장은 표지, 마지막 장은 마무리 + 팔로우 유도로 구성
- 하단 브랜드명: @[브랜드명]
- 출력:
  - 카드 이미지: output/cards/card_01.png부터 순서대로
  - 인스타그램 게시글 문구: output/cards/caption.md
    (본문 텍스트 + 해시태그 포함)
```

### Step 6 — 주제 직접 입력해서 카드뉴스 제작

```
테스트 결과 [minimal/colorful/dark] 스타일로 진행하기로 했어.

주제: "[주제를 입력하세요]"

이 주제로 카드뉴스를 만들어줘.
- style.md 스타일을 정확히 따를 것
- 핵심 내용을 5~10장 사이로 적절하게 구성해줘
- 각 카드는 한 가지 메시지만 담고, 텍스트는 핵심만 임팩트 있게
- 첫 장은 표지, 마지막 장은 마무리 + 팔로우 유도로 구성
- 하단 브랜드명: @[브랜드명]
- 출력:
  - 카드 이미지: output/cards/card_01.png부터 순서대로
  - 인스타그램 게시글 문구: output/cards/caption.md
    (본문 텍스트 + 해시태그 포함)
```

### Step 7 — CLAUDE.md 정리 (세션 인계용)

```
지금까지 한 모든 작업을 CLAUDE.md 파일에 정리해줘.

포함할 내용:
1. 이 프로젝트가 뭘 하는 프로젝트인지 (한 줄 설명)
2. 폴더 구조와 각 파일의 역할
3. 카드뉴스 만드는 전체 프로세스 (단계별)
4. 새로운 주제나 파일로 카드뉴스 만들 때 어떻게 하면 되는지

Claude Code의 새로운 세션에서 이 파일만 보고도
완벽히 동일한 프로세스로 바로 작업할 수 있도록 작성해줘.
```

### Step 8 — Unsplash 배경 이미지 기능 추가 후 CLAUDE.md 업데이트

```
방금 추가한 Unsplash 배경 이미지 기능을 CLAUDE.md에 업데이트해줘.
변경사항과 API 키 설정 방법도 포함해줘.
```

---

## 팁

- **CLAUDE.md**를 항상 최신 상태로 유지할 것 → 새 세션에서 바로 이어서 작업 가능
- **input 폴더**에 txt, md, pdf 파일 넣으면 자동으로 내용 파악해서 카드뉴스 생성
- Unsplash API 키는 `UNSPLASH_ACCESS_KEY` 환경변수로 설정
- 스타일별 MD 파일을 잘 관리하면 여러 브랜드에 재사용 가능
