# 유튜브 영상 편집용 사운드 효과 가이드

> 유튜브 영상 편집 효과음 카테고리별 사용 가이드

## 효과음 카테고리

| 번호 | 카테고리 | 사용 장면 |
|------|---------|---------|
| 1 | **슉 (Whoosh)** | 씬과 씬 사이, 자막 등장. 모든 전환에 다 넣지 말고 분기점 위주 |
| 2 | **임팩트, 강조** | 충격적 폭로, 결정적 순간, 자막을 크게 띄울 때 |
| 3 | **뽕, 팝** | 자막이나 이미지가 톡톡 튀어오르듯 등장할 때. 가벼운 강조용 |
| 4 | **클릭, 딩** | 선택, 결정, 알림 순간. 짧고 명확한 신호 |
| 5 | **긴장감** | 결과 공개 직전, 갈등 고조, 비밀이 밝혀지기 전 |
| 6 | **결과 공개, 격투** | 갈등이 폭발하는 순간, 폭로, 대립 장면 |
| 7 | **아이러니, 비꼼** | 황당한 상황, 잘못된 판단, 비꼬는 분위기 |
| 8 | **분위기 반전** | 장면이나 분위기를 갑자기 바꿀 때, 회상에 들어갈 때 |
| 9 | **개그 코믹** | 코믹한 상황, 황당한 반응, 가벼운 실패 장면 |
| 10 | **감성 타이핑·자연소리** | 배경 분위기 환경음. BGM과 겹쳐 사용 가능 |

## 활용 팁

- **기본 추천 포인트**: 인트로 / 회상 진입 / 충격적 폭로 / 클라이맥스 / 마무리
- **회상 시작**: reverse whoosh → **회상 종료**: cinematic whoosh
- **결과 공개 전**: drum roll → bomb drop / cinematic impact 순으로 빌드업
- **감성 자연소리** 볼륨: BGM 대비 -15dB ~ -20dB 추천
- **같은 효과음을 한 영상에서 3번 이상 반복 사용 금지**
- 50~70대 시청자 대상: 코믹 효과음 신중하게 사용
