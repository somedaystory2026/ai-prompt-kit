# 캡컷 자동 편집 에이전트 (FastAPI + HTML)

> **추천 AI**: Claude Code

## 프롬프트

```
[정체성]
자동화 영상 편집 도구 빌드 파트너.

[만들 것]
한국어 토킹 영상 자동 편집기 "캡컷 에이전트"
입력 mp4/mov → 출력 캡컷 드래프트 (무음·잔말·NG 컷 + 단어별 자막)
형태: 로컬 웹 (FastAPI + 정적 HTML 1장)

[핵심 원칙]
★ 전체 대본(script) 먼저 추출 → 그걸로 NG/자막 결정
★ 검증 = 사용자가 캡컷에서 직접 재생. 빌드 성공 ≠ 검증

[Step 0: OS 감지 + 환경 점검]
python3 -c "import platform; print(platform.system(), platform.machine())"
▶ Darwin arm64 → 트랙 A (Mac M칩, mlx-whisper)
▶ Windows AMD64 → 트랙 C (faster-whisper, MP4 export 보너스)

[빌드 적층 — 각 단계 캡컷에서 실제 재생 검증 후 다음]
1단: silence_detect + build_draft → 점프컷 드래프트 (핵심 검증)
2단: FastAPI + 정적 HTML (drag/drop + SSE stepper)
3단: whisper Transcript(세그먼트+단어) + 세그먼트 단위 자막
4단: filler_ng + cuts (잔말/NG 통합 컷) + 결과 카드 transcript 노출
5단: 영상 프리뷰 + 보존 구간 사용자 마킹 ([/]키 단축키)

[런타임 단계 — 영상 1개 처리 시 SSE 이벤트]
silence → asr → filler → draft+자막
※ ASR은 asyncio.Lock으로 직렬화 (numba 비안전, 동시 호출 시 segfault)
※ 단계당 최소 0.5s 강제 지연

[디자인 톤]
Linear/Vercel/Notion 모노 + 그린 #22c55e 1색 액센트만.
다크 #0a0a0a + 카드 #161616.
❌ 그라데이션, 좌측 컬러 띠, 큰 ✓, 박스 안에 박스
✓ 6px 점 + 글로우, 3컬럼 grid 통계, 1px hr 구분, 코드 칩
```
