1. 로우컷 (Low Cut) — 80~120Hz 보컬에 필요 없는 저음 정리. 너무 많이 자르면 얇아지고, 너무 적게 자르면 뭉침. 2. 로우미드 컷 (200~400Hz) — 약 -2dB 이 영역이 많으면 보컬이 답답하고 뿌옇게 들림. 이것만 해도 답답한 느낌이 확 줄어듦. 3. 프레즌스 (3kHz) — 살짝 부스트 보컬의 선명도와 존재감. 살짝만 올려도 보컬이 앞으로 튀어나옴. 4. 하이쉘프 (6kHz+) — +2~3dB 에어감과 열린 느낌. ⚠️ 수노 음원은 고음역 디지털 노이즈 있음. 들으면서 조금씩 조절. 핵심 정리: 로우컷 → 로우미드 컷 → 프레즌스(3kHz) → 하이쉘프(부스트)

인스타그램 카드뉴스 자동 제작 (HTML+Playwright)
NEW · Claude Code
Claude Code로 인스타그램 카드뉴스를 HTML/CSS로 만들고 Playwright로 PNG 이미지로 저장하는 방법으로 하고 싶어. 인스타그램 카드뉴스용 디자인 스타일을 3가지 추천해줘. - 스타일 중 하나는 레드 계열 다크 톤 + 중앙 정렬 느낌으로 - 나머지 둘은 분위기가 서로 다른 스타일로 자유롭게 추천해줘 각 스타일에 포함할 내용: 1. 색상 팔레트 (배경, 텍스트, 강조 색상) 2. 폰트 (제목, 본문) / 3. 레이아웃 / 4. 카드 구성 요소 5. 카드 비율: 4:5 / 6. 분위기 키워드 3개 각각 별도 마크다운 파일로 저장해줘. 실제 제작 명령: 스타일이 확정되면 주제를 주면 카드뉴스를 만들어줘. - style.md 스타일을 정확히 따를 것 - 핵심 포인트를 파악해서 5~10장 사이로 적절하게 구성 - 첫 장은 표지, 마지막 장은 마무리 + 팔로우 유도 - 출력: output/cards/card_01.png~ / output/cards/caption.md 작업이 끝나면 CLAUDE.md에 정리해줘. 주제: [입력] / 브랜드명: @[입력]
모션 그래픽 3종 (Remotion)
Claude Code + Remotion
1. 네온 삼각형 애니메이션 화면 중앙에서 네온 핑크색 삼각형이 회전하며 서서히 커지면서 나타나게 해줘. 세 꼭짓점에서 민트색 빛의 줄기 3개가 동시에 뿜어져 나와 원형으로 빠르게 휘감아. 스피드 램프 효과로 회전 속도가 빨라졌다 느려지기를 반복. 마지막에는 삼각형이 반대로 회전하면서 모든 빛줄기를 안으로 끌어당기고 사라지는 연출. 2. 글로우 공전 박스 (Remotion) remotion 모범 사례로 알약 모양의 직사각형 박스를 만들어줘. 박스 주위에 글로우(빛 번짐)가 궤도를 돌고 있어야 해. 부드럽게 빙글빙글. 박스 안에 텍스트 없음. 슈퍼 프리미엄 인포그래픽/UI 모션 스타일. 3. 7개 박스 순차 등장 (Remotion) 깔끔한 애플 감성의 직사각형 박스 7개 세트. 각 박스는 블러 인 + 페이드 인 + 부드럽게 위로 슬라이드 업으로 등장. 각 2초. 박스 제목: 효과 1~6 / 보너스 효과
