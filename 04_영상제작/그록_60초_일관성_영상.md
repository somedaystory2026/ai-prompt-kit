# 그록 60초 일관성 영상 제작

> **추천 AI**: Grok

## 1단계 — 이야기 생성

```
시각적 일관성을 유지하기 위해 정확히 두 명의 주인공이 등장하는
60초 분량의 짧은 이야기를 만들어줘.
이야기는 로맨스 요소가 가미되어야 하며,
시각화하기 쉬운 단순한 언어로 작성해야 돼.

구조 및 요구 사항:
- 분량: 낭독 시 60초 이내
- 등장인물: 이야기 전체에 걸쳐 오직 두 명의 주인공만
- 훅: 시청자의 몰입을 유지하기 위해 약 5초마다 작은 긴장감 요소 추가
- 톤: 매우 영화적, 추상적인 생각보다는 명확한 신체적 행동 중심
- 이야기 구성: 시작 → 고조되는 긴장감 → 전환점(반전 또는 진실) → 만족스러운 결말
- 장르: 로맨스가 자연스럽게 어우러짐
```

## 2단계 — 장면별 프롬프트 분할

```
방금 만든 이야기를 바탕으로 명확하고 포괄적인 장면별 프롬프트로 나눠줘.
각 장면은 캐릭터의 행동, 배경, 분위기를 포함하여 시각적으로 어떤 일이 일어나고 있는지 설명해야 합니다.

- 각 장면은 5초에서 8초 사이의 화면 시간을 유지해줘.
- 모든 장면이 다음 장면으로 자연스럽게 이어져서,
  최종 영상이 하나의 연속적인 영화 시퀀스처럼 느껴지도록 해줘.
- 이야기에 원래 없던 새로운 등장인물이나 배경을 도입하지 마.
- 설명은 AI 비디오 생성기가 이해하기 쉽도록 직관적이고 시각적이며 쉬워야 해.
```
