# 해부학 3D 영상 프롬프트 생성기

> **추천 AI**: ChatGPT 또는 Claude

## 프롬프트

```
내가 프롬프트를 입력하면, 너의 첫 번째 임무는 나에게 단 하나의 질문만 던지는 거야:
"음식이나 음료의 이름을 입력해 주세요."

내가 이름을 제공하면(예: "우유"), 너는 아래의 모든 프롬프트를 즉시 생성하며,
모든 내용에서 "우유"를 내가 제공한 음식이나 음료로 자동 교체해야 해.
다시 설명하거나 재생성하지 말고, 즉시 최종 프롬프트 생산을 시작해줘.

이미지 생성 지침: 전체 배경이 초록색인 상세한 3D 렌더링.

주요 4장면:
1. 자연스러운 사람이 음식이 담긴 잔을 입 근처에 대고 있는 모습 +
   내부 장기(뇌, 폐, 심장, 위, 장)가 사실적이고 상세하게 노출된 초현실적 해부학 일러스트레이션
2. 위에만 집중하여 음식이 위장 속으로 부어지는 모습 (사실적인 의료 디테일)
3. 장에서 영양소를 흡수하는 모습 - 빛나는 영양소 입자들이 혈류로 들어가는 미세 현미경 스타일
4. 영양소들이 전신으로 퍼져나가는 긍정적인 효과 - 장기와 근육이 에너지로 밝게 빛나는 모습

추가 이점 이미지 프롬프트 (3~4개): 뇌 기능 향상, 근육 영양 공급, 장 건강, 에너지 상승
부작용 이미지 프롬프트 (3~4개): 위장 자극, 알레르기 반응, 혈당 스파이크, 피로/저에너지

영상 프롬프트: 위의 모든 장면에 대해 5~10초 길이의 3D 영화적 영상 프롬프트를 작성해줘.
카메라 움직임, 애니메이션 동작, 사실적인 조명을 설명해야 해.

출력 구성:
1. 📸 이미지 프롬프트
2. 🎥 영상 프롬프트

이미지는 만들지 마.
```
