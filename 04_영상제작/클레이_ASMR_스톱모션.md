# 클레이 ASMR 스톱모션 3단계

> **추천 AI**: ChatGPT 또는 Claude

## STEP 1 — 주제 및 캐릭터 기획

```
클레이 스톱모션 ASMR 영상을 만들려고 해.
잔잔한 느낌이 들고, 보는 사람이 행복한 감정을 느낄 수 있는
힐링 주제 10가지를 뽑아줘.
그리고 각 주제마다 어울리는 귀여운 주인공 동물 캐릭터를 2마리씩 추천해 줘.
```

## STEP 2 — 캐릭터 일관성 세팅

```
위 주제 중에 [n]번째 주제로 할게.
먼저 영상의 일관성을 유지하기 위해 동물 캐릭터들의 외형을 세팅할 거야.
내가 이미지 생성 AI에 바로 넣을 수 있도록,
귀여운 2마리 캐릭터의 영문 이미지 프롬프트를 각각 작성해 줘.
(이미지 생성 금지, 영문 프롬프트 텍스트만 출력)
```

### 클레이 질감 고정 규칙 (모든 프롬프트에 반드시 포함)

```
- 배경: isolated on a seamless, clean light grey background void
- 형태: highly adorable, chubby, and charming miniature proportions
         with rounded, soft shapes
- 재질: pristine, non-greasy, satin-matte finish polymer clay
- 표면: velvety, mochi-like surface grain that absorbs light
         (진짜 동물의 털 표현 절대 금지)
- 수작업 디테일: visible soft manual undulations and gentle tool marks from hand-sculpting
- 조명: warm, side-directional cinematic tungsten lighting casting soft, rounded shadows
```

## STEP 3 — 시네마틱 멀티 앵글 스토리보드 생성

```
이제 확정된 캐릭터와 주제를 가지고,
실제 스톱모션 애니메이션을 만들기 위한 시네마틱 멀티 앵글 스토리보드를 짤 거야.

먼저 단 1장의 완벽한 [마스터 씬(배경) 이미지 프롬프트]를 영어 전문으로 작성해 줘.
(캐릭터들이 주제에 맞는 배경 안에서 전체적인 분위기를 보여주는 와이드 샷)

이후 스토리보드 작성 규칙:
1. 앵글 다양하게: 1컷-와이드, 2컷-손 클로즈업, 3컷-미디엄... 번갈아 구성
2. 액션은 무조건 클로즈업: 잡거나·붓거나·섞는 동작은 Paws(손) 클로즈업
3. 시간적 연속성: 동작 순서 논리적으로 이어지게
4. 축약 절대 금지: 모든 프롬프트 칸은 영어 전문으로 채울 것

비디오 프롬프트 작성 규칙:
[카메라 앵글] + [누가/무엇을/어떻게 움직이는지 심플한 묘사]
+ [12fps 스톱모션 스타일] + [ASMR 사운드]만 작성.
배경이나 질감 묘사는 싹 빼고 오직 위 내용만.

출력 형식 (표):
| 컷 번호 | 세부 행동 및 앵글 (한국어) | 이미지 생성 프롬프트 (Full Text 영어) | 비디오 생성 프롬프트 (Full Text 영어) |
```
