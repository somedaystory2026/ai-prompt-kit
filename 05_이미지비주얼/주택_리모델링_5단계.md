# 주택 리모델링 5단계 프롬프트

> **추천 도구**: AI 이미지·영상 생성기

## 프롬프트 5종

```
1. 주택 생성
럭셔리한 주택을 9:16 화면 비율로 만들어줘

2. 폐허가 된 집
오랜 세월 방치된 폐가 느낌으로 꾸며줘. 마당엔 잡초가 사람 키만큼 자라 있고,
주변엔 쓰레기 더미가 가득해. 벽은 여기저기 금이 가고 낙서가 잔뜩 그려져 있어야 해.
창문은 다 깨지고 문은 녹슬어서 금방이라도 무너질 것 같은 분위기랄까?
최대한 처참하고 황량한 느낌을 살려줘.

3. 보수 완료 (기초 상태)
이 집을 아주 깔끔하게 정리해줘. 쓰레기는 하나도 없이 싹 치우고,
갈라진 벽이랑 망가진 창문도 완벽하게 수리된 상태야.
가구나 유리창은 다 빼고, 딱 페인트칠만 하면 되는 '텅 빈 도화지' 같은 공간으로 만들어줘.

4. 철거 및 청소 타임랩스
집을 고치는 과정을 타임랩스 영상으로 보여줘. 인부들이 바쁘게 움직이면서 쓰레기를 실어
나르고, 무성했던 풀을 베어내는 장면이야. 고압 세척기로 벽을 닦아내고 마지막엔 딱
페인트칠하기 직전의 깨끗한 상태가 되는 과정을 역동적인 공사 현장 느낌으로 담아줘.

5. 인테리어 완성 타임랩스
집 수리 타임랩스 영상이야. 작업자들이 페인트칠을 하고, 가구랑 장식 소품을 배치하고,
조명까지 달아서 마지막 장면이랑 똑같이 만드는 모습이야.
텅 빈 공간이 멋진 야외 공간으로 바뀌는 과정을 빠른 화면으로 보여줘.
```
