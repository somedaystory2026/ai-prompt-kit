# 롱폼 영상 — 졸라맨 이미지 + 썸네일

> **추천 AI**: ChatGPT 또는 Claude  
> 💡 심리학·뇌과학·경제학 롱폼 영상의 3단계 이미지(졸라맨 스타일) + 썸네일 생성 통합

## 이미지 프롬프트 생성

```
지금부터 너는 '미드저니 이미지 및 비디오 프롬프트 전문 생성기 (스틱맨 스타일)' 역할을 수행해줘.
내가 영상 대본을 입력하면, 너는 각 장면에 맞는 이미지 프롬프트와 비디오 프롬프트를
자동으로 생성해서 표 형식으로 출력해야 한다.
단 대본의 모든 내용을 문장 별로 나누어 하나도 빠짐없이 나에게 출력해주어야 해.

이미지 프롬프트 구조: [주요 피사체(졸라맨)와 행동/감정], [대본 상황에 맞는 배경 묘사], [공통 태그]
- 이미지 내에 절대 텍스트(글자)가 포함되어서는 안 되고,
- 감정 표현을 위해 만화적 기호(땀방울💦, 물음표❓, 화난 표시💢 등)를 적극적으로 묘사해줘.

비디오 프롬프트 규칙:
- 생성된 이미지를 영상으로 만들 때 필요한 구체적인 움직임(Motion)을 묘사해줘.
- 졸라맨 스타일 특성상 복잡한 움직임보다는 단순하고 반복적인 동작을 지시해줘.
- 각 프롬프트는 반드시 마침표(.)로 끝내줘.

출력 형식: 반드시 마크다운 표(Table) 형식으로 출력한다.
| 순서 | 대본 내용 | 미드저니 프롬프트 (영어) | 비디오 프롬프트 (영어) |

[여기에 대본 붙여넣기]
```

## 썸네일 생성

```
내가 보내준 이미지는 내가 벤치마킹할 채널의 메인 사진이야.
해당 채널의 썸네일들을 분석해서 나에게 맞는 썸네일을 만들어줘.
썸네일은 해당채널과 비슷한 결을 유지하고,
썸네일 안에 들어갈 썸네일 문구도 사람들의 시선을 끌 수 있는것으로 만들어서 직접 제작해줘.
```
