# Google Flow 이미지 + Grok 영상 자동 생성

> **추천 도구**: AI연구소 Studio (크롬 확장 프로그램)  
> 💡 Google Flow에서 이미지를 자동으로 생성하고, Grok Imagine에서 영상을 자동으로 생성합니다.

## 오토플로우 사용법 (이미지 자동 생성)

1. Google Flow 페이지 미리 열어두기: `labs.google/fx/tools/flow`
2. 확장프로그램 아이콘 클릭 → 사이드패널 열기
3. **오토플로우** 탭 선택
4. 프롬프트를 한 줄에 하나씩 입력
5. **작업 시작** 클릭

### 주요 기능

- **영문 자동 번역**: 체크하면 한국어 프롬프트를 영어로 번역해 Flow에 전송 (이미지 품질 향상)
- **스타일 프롬프트**: 모든 프롬프트 끝에 자동으로 붙는 스타일 키워드  
  예) `지브리 스타일, 수채화` → 모든 프롬프트가 "~~, 지브리 스타일, 수채화" 형태로 전송
- **생성 간격**: 랜덤 10~15초 설정 (너무 짧으면 Flow에서 차단될 위험)
- **자동 저장**: 이미지 저장 형식 = `접두사_순번_프롬프트앞15자.png`

### 참조 이미지 활용 (@[이름] 태그)

```
@[햄토리]가 학교에서 달린다
@[교실]에 들어선 @[햄토리]
```

## 오토그록 사용법 (영상 자동 생성)

- 이미지 + 프롬프트를 입력하면 Grok Imagine에서 영상을 자동 생성
- 텍스트만으로도 영상 생성 가능
- **Grok 사이트는 반드시 한국어 설정**으로 유지
