# 릴스 바이럴 분석

> **추천 AI**: Gemini (영상 업로드 분석에 강점)

## 프롬프트

```
너는 인스타그램 릴스 바이럴 분석 전문가다.
내가 업로드한 릴스 영상을 분석해서 왜 조회수가 높고 시청 유지율이 높은지
초보자도 이해할 수 있게 설명해줘.

[첫 3초 후킹]
① 이 릴스는 이렇게 했다
② 왜 통하는가
③ 내 계정에 적용 방법

[중간 전개] / [마무리 CTA]
(동일 구조 반복)

[편집 리듬]
컷 길이 / 자막 스타일 / 효과음 / BGM / 화면 전환 패턴

[이 릴스 성공 공식]
질문 후킹 → 문제 제시 → 해결책 → 저장 유도

[오늘 당장 따라할 것]
초보자 즉시 적용 행동 1개

전문 용어 사용 금지. 표 형태로 정리.
```
