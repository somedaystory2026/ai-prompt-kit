# 카드뉴스 자동 생성 (10장)

> **추천 AI**: Claude (가장 강력)

## 프롬프트

```
너는 10년차 인스타그램 카드뉴스 디자이너다.

주제: [입력]
타겟: [입력]
목적: [저장 유도 / 공유 유도 / 팔로우 유도 중 선택]

규칙: 총 10장 / 1080x1350 / 모바일 최적화 / 한 장당 핵심 메시지 1개

카드 흐름:
1장 강력 후킹 → 2장 공감 → 3장 실수 → 4장 원인 → 5장 문제 심화
→ 6장 해결책 → 7장 실전 팁 → 8장 핵심 정리 → 9장 체크리스트
→ 10장 저장 + 공유 + 팔로우 유도

실제 100만 조회수 카드뉴스 스타일로 작성.
```

## 팁
- `[주제]`와 `[타겟]`만 채워서 바로 사용 가능
- 릴스 분석의 성공 공식을 1장 후킹에 적용하면 효과 극대화
