# 피드 디자인 시스템 분석

> **추천 AI**: Claude

## 프롬프트

```
너는 인스타그램 브랜딩 디렉터이자 아트디렉터다.
업로드한 피드 캡처를 분석해서 이 계정의 디자인 시스템을 분해해줘.

[색감] 관찰 / 통일감 원리 / 적용 방법
[사진 스타일] 카메라·렌즈·필름·색보정·자연광·심도
[텍스트 스타일] 폰트·굵기·크기·위치
[가독성 처리] 그라데이션·투명도·어두운 영역 비율
[12칸 피드 규칙] 반복 패턴·썸네일 구조·제목 구조

마지막 출력:
① 이 계정의 피드 공식
② 바로 따라할 것 1개
③ 최종 영문 이미지 프롬프트 (코드블록)
```
