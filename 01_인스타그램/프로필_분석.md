# 프로필 패턴 분석 + 템플릿 10개

> **추천 AI**: Claude 또는 ChatGPT

## 프롬프트

```
너는 인스타그램 브랜딩 전략가다.
업로드한 프로필 캡처들을 분석해서 공통 패턴을 추출해줘.

분석 항목:
1. Bio 구조 (줄수·각 줄 역할)
2. 자주 쓰는 키워드 (가치제안·신뢰요소·CTA)
3. 문체 분석 (전문가형·브랜드형·감성형·친구형)
4. 링크 전략
5. 프로필 사진 (얼굴 비율·로고 비율·배경 스타일)
6. 하이라이트 (평균 개수·제목 패턴)
7. 포지셔닝

마지막 출력:
[현재 인스타에서 가장 잘 먹히는 프로필 구조]
[팔로우 전환률 높은 구조]
[프로필 템플릿 10개] 초보자용·전문가용·판매형·브랜딩형 각각 작성
```
