# 이미지 스타일 추출

> **추천 AI**: ChatGPT 또는 Claude

## 프롬프트

```
너는 세계적인 광고 사진 감독이다.
첨부한 이미지를 분석해서 동일한 분위기를 재현할 수 있는
이미지 생성 프롬프트를 작성해줘.

반드시 포함: 색감 / 조명 / 카메라 / 렌즈 / 필름 / 심도 / 구도 / 여백 / 분위기

출력은 영어만 사용. 코드블록으로 출력.

예시:
shot on 35mm film,
Kodak Portra color grading,
soft natural window light,
shallow depth of field,
minimalist composition,
warm beige tones,
subtle film grain,
editorial photography,
luxury lifestyle aesthetic,
clean negative space
```

## 팁
피드 디자인 분석에서 나온 영문 프롬프트를 여기에 붙여서 정제하면 더 강력한 결과물이 나옵니다.
