# Suno AI 통합 마스터 프롬프트

> **사용 도구**: Suno AI  
> 💡 Suno 마스터 시스템 + 기본 마스터 + K-Pop 특화 + 프로듀싱 전문가 프롬프트 4개 통합

## 핵심 작동 원칙

- **자연어 기반 영문 서술** (구조화 태그 [BPM=120] 등 사용 금지)
- **Style of Music 최적 길이**: 150~250자
- 가장 중요한 요소(장르·분위기)를 **맨 앞**에 배치
- 부정 지시 사용 금지 ([피아노 없음] 등 → AI 혼란)
- 아티스트 레퍼런스: **1~2명만** (3인 이상 → 방향성 희석)

## Style of Music 5단계 구조 공식

```
[1단계: 핵심 정체성]
장르(하위 장르까지) + 분위기(구체적 장면 묘사) + 레퍼런스 1~2명

[2단계: 사운드 & 악기]
핵심 악기 3~5개의 역할과 상호작용을 자연어로 묘사 + 음향 질감 형용사

[3단계: 보컬]
[Male vocal] 또는 [Female vocal] + 음색/감정 표현 스타일 구체 명시

[4단계: 전개]
곡의 다이내믹스 흐름 (잔잔 → 폭발, 점층적 긴장감 등)

[5단계: 품질 태그]
[Audio quality improved to masterpiece]
```

## 출력 형식

```
[Style of Music]:
(영문 150~250자)

[Lyrics]:
[Intro]
[Verse 1]
[Pre-Chorus]
[Chorus]
[Verse 2]
[Pre-Chorus]
[Chorus]
[Bridge]
[Final Chorus]
[Outro]
```

## K-Pop 특화 규칙

```
- 언어 비율: 한국어 70% : 영어 훅 30%
- 후렴은 틱톡에서 유행할 정도로 중독적
- 20대 여성 화자 시점 (기본값)
- 영어 훅은 후렴 핵심 라인에 배치
```

## 가사 작성 7대 규칙

1. 실제 사람이 말하는 것처럼 자연스럽게
2. 클리셰 금지: `dream, shining, sky, tonight, forever, tapestry`
3. 후렴은 짧고 중독성 있게 (틱톡·숏폼 바이럴 기준)
4. 각 Verse는 스토리가 진행되도록
5. 감정선이 Intro → Final Chorus로 점층적으로 커지도록
6. Suno가 노래하기 쉬운 짧은 문장 위주
7. 전체 **3,000자 이내** 유지

## 연출 태그 (가사 내 삽입)

| 태그 | 효과 |
|------|------|
| `[SFX]` | 효과음 삽입 포인트 |
| `[Emotional]` | 감정 폭발 구간 |
| `[Energy High]` | 에너지 최고조 |
| `[Energy Low]` | 차분하게 내려앉는 구간 |
| `[Whispers]` | 속삭임 보컬 |
| `[Instrumental]` | 악기 솔로 구간 |
| `[Guitar Solo: 설명]` | 특정 악기 지시 |
| `[Fade-Out]` | 아웃트로 페이드 |
