# AI 음악 프롬프트 사운드 단어 100개

> Suno AI에 사용할 수 있는 사운드 단어 빠른 참조 가이드

## 1. 악기 / 사운드 소스

| 단어 | 설명 |
|------|------|
| `Warm analog Rhodes` | 따뜻하고 빈티지한 전자피아노 사운드 (R&B, 네오소울) |
| `Electric piano` | 부드럽고 현대적인 전자 피아노 톤 |
| `Grand piano` | 클래식하고 선명한 피아노 사운드 |
| `Acoustic guitar` | 자연스럽고 따뜻한 어쿠스틱 기타 |
| `Fingerstyle guitar` | 손가락으로 연주하는 섬세한 기타 스타일 |
| `Jangly guitar` | 반짝이고 밝은 인디팝 기타 사운드 |
| `Ambient guitar` | 리버브가 넓게 걸린 공간감 있는 기타 |
| `Synth pad` | 배경을 채우는 넓고 부드러운 신스 |
| `Supersaw synth` | EDM에서 사용하는 두껍고 화려한 신스 |
| `808 bass` | 힙합과 트랩에서 많이 사용하는 강한 저음 |
| `Sub bass` | 매우 깊고 묵직한 저역 베이스 |
| `String ensemble` | 여러 대의 바이올린이 만드는 스트링 사운드 |
| `Orchestral brass` | 영화 음악 스타일의 강한 브라스 |
| `Choir vocals` | 합창 스타일의 보컬 사운드 |

## 2. 리듬 / 드럼 스타일

| 단어 | 설명 |
|------|------|
| `Boom bap groove` | 올드스쿨 힙합 스타일의 드럼 리듬 |
| `Trap hi-hats` | 빠르게 반복되는 트랩 스타일 하이햇 |
| `Punchy kick` | 단단하게 치고 나오는 킥 드럼 |
| `Four-on-the-floor` | 하우스 음악의 기본 킥 패턴 |
| `Swing groove` | 재즈 느낌의 흔들리는 리듬 |
| `Lo-fi drum groove` | 살짝 거칠고 빈티지한 드럼 사운드 |
| `Latin groove` | 라틴 음악 특유의 리듬 패턴 |
| `Driving rhythm` | 에너지가 강하게 밀어붙이는 리듬 |
| `Festival drums` | EDM 페스티벌 스타일의 강한 드럼 |

## 3. 사운드 질감 / 프로덕션 스타일

| 단어 | 설명 |
|------|------|
| `Analog warmth` | 아날로그 장비 느낌의 따뜻한 사운드 |
| `Tape saturation` | 테이프 장비에서 나는 빈티지 질감 |
| `Vinyl texture` | LP 레코드 느낌의 거친 질감 |
| `Lo-fi texture` | 일부러 음질을 거칠게 만든 스타일 |
| `Wide stereo` | 좌우로 넓게 퍼지는 사운드 |
| `Sidechained bass` | 킥에 맞춰 펌핑되는 베이스 |
| `Radio-ready mix` | 상업적인 완성도를 가진 사운드 |
| `Organic production` | 실제 연주 느낌이 살아있는 사운드 |
