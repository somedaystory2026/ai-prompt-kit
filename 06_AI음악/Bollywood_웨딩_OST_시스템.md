## 핵심 개념: 4단계 감정 구조

Bollywood OST는 단순 음악이 아니라 **"장면 → 감정 → 폭발 → 해방"**의 4단계 감정 구조로 움직입니다. 웨딩 영상이라면 여기에 **사랑(romance) / 눈물(emotion) / 축제(celebration) / 에너지 폭발(dance climax)** 4가지 감정이 섞입니다.

## OST 5단계 구조 (영화/웨딩 공통)

| 단계 | 일반 OST | 웨딩 OST |
|------|----------|----------|
| 1. INTRO | 장면 설정 (비 오는 거리 / 결혼식 / 이별 / 전야) | 신부 등장 슬로우 모션, piano + flute, 눈물 감정 시작 |
| 2. VERSE | 감정 서사 (사랑/운명/가족/갈등) | 사랑 서사 (과거 회상), 부드러운 strings + vocal |
| 3. PRE-CHORUS | 긴장 상승 (드럼+스트링 상승, 폭발 직전) | dhol 점점 들어옴, 감정 상승 |
| 4. CHORUS | 폭발 (dhol+shehnai+full orchestra, 명대사 훅) | 웨딩 폭발 (dhol+tabla+shehnai+orchestra, 모두 춤추는 순간) |
| 5. FINAL | 클라이맥스 (전악기 폭발, 합창+드럼) | 클럽 수준 폭발 피날레 (EDM+Bollywood fusion drop) |

INTRO 예시 대사: `"Rain falls on empty streets… she is waiting…"`
CHORUS 훅 예시: `"Tu hi mera naseeb hai…"` (너는 내 운명이야…)
웨딩 CHORUS 훅 예시: `"Dil se dil mila…"` (heart meets heart)

## 핵심 사운드 라이브러리

- **감정 계열**: Strings(violins, 감정 상승) / Piano(intro 슬픔) / Choir(클라이맥스)
- **리듬 계열**: Dhol(웨딩/축제 에너지) / Tabla(로맨틱 긴장) / Drum kit(모던 EDM 퓨전)
- **멜로디 계열**: Shehnai(웨딩 감정) / Flute(향수/사랑) / Sitar(운명/영성)

## 영상 + 음악 동시 생성 시스템 (웨딩 전용)

> 핵심 아이디어: "OST는 음악이 아니라 영상의 엔진이다"

### 입력 변수
- 주제: 웨딩 / 사랑 / 감정 / 축제
- 톤: 로맨틱 / 드라마 / 폭발 / 감성
- 장면: 인도식 결혼식 / 현대식 웨딩 / 영화 스타일

### 영상 구조 (15~60초 기준)

| 구간 | 장면 | 음악 |
|------|------|------|
| 0~5초 (INTRO) | 신부 슬로우 모션 입장, 꽃잎/조명/슬로우 줌 | piano + flute |
| 5~15초 (LOVE BUILD) | 눈 맞춤/손잡기/감정 회상 | soft strings + vocals |
| 15~30초 (DANCE BUILD) | 사람들이 하나씩 춤 시작 | dhol 들어오기 시작 |
| 30~45초 (CLIMAX DROP) | 전원 춤+군중+카메라 흔들림, fireworks/lights | full Bollywood EDM fusion |
| 45~60초 (FINAL SHOT) | 신랑신부 슬로우 모션 | 감정 fade out or loop |

## 100만 조회수 나는 이유 (웨딩 VIRAL FORMULA)

1. **감정 4단계 구조**: 사랑 → 긴장 → 폭발 → 해방
2. **시각 + 음악 동기화**: "음악 = 장면 진행 엔진"
3. **반복 가능한 후렴**: Shorts/Reels 자동 루프
4. **문화 코드(Bollywood)**: 전 세계적으로 이해 가능한 감정 구조
