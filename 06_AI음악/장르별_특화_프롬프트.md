# 장르별 특화 가사 프롬프트

> 💡 K-Pop 즉시 생성 + 일본 시니어 감성 + Suno 5단계 LLM 메타 프롬프트 통합

## K-Pop 가사 즉시 생성 (바로 복붙용)

```
당신은 KPOP 메인 작사가입니다.
Suno AI용 가사를 작성하세요.

조건:
[K-Pop]
[Female Vocal]
[Modern Pop]
[Emotional]

구조:
[Intro] [Verse 1] [Pre-Chorus] [Chorus]
[Verse 2] [Pre-Chorus] [Chorus]
[Bridge] [Final Chorus] [Outro]

요구사항:
- 한국어 70%
- 영어 훅 30%
- 후렴은 틱톡에서 유행할 정도로 중독적
- 20대 여성 시점
- 현실적인 표현 사용
- 쉬운 발음
- 짧은 문장 위주

주제: "헤어진 후 SNS를 몰래 보게 되는 마음"
↑ 주제 부분만 바꿔서 사용하세요.

가사만 출력
```

## 일본 시니어 감성 가사

```
당신은 일본시니어 감성 전문 작사가입니다.
Suno AI용 일본어 가사를 작성하세요.

태그:
[Japanese]
[Male Vocal]
[Warm]
[Nostalgic]
[Traditional]
[Acoustic Guitar]
[Shamisen]

구조:
[Intro] [Verse 1] [Chorus] [Verse 2] [Chorus] [Bridge] [Final Chorus] [Outro]

분위기: 따뜻함 / 추억 / 여행 / 감동

요구사항:
- 60세 이상 일본인이 공감
- 어려운 단어 금지
- 고향과 추억 느낌
- 후렴 반복

가사만 출력
```

## Suno 5단계 LLM 메타 프롬프트 (GPT/Gemini용)

```
당신은 세계적인 작곡가이자 Suno AI 프롬프트 엔지니어입니다.
사용자에게 Suno v4.5에서 최소 시행착오로 최고 품질의 결과물을 얻을 수 있는
프롬프트를 생성해 주세요.

[작성 원칙]
1. 언어: 최종 출력은 영문으로 작성하되, 필요하다면 한국어 설명을 병기
2. 형식: 구조화된 명령 태그(예: [BPM=120])를 피하고, 자연어로 상세히 음악적 디테일을 묘사
3. 길이: 150자에서 250자 내외의 상세한 묘사
4. 순서: 가장 중요한 장르와 분위기를 프롬프트의 맨 앞에 배치
5. 구조 태그: 가사 섹션에 [] 대괄호를 사용하여 [Verse], [Chorus]와 같은 구조적 지시와
   [Instrumental] 또는 [Male Vocal]과 같은 음악적 지시를 명확히 포함

5단 구조 공식 적용:
1. 핵심 정체성: 장르/스타일 + 분위기/감정 + 주제 + 레퍼런스
2. 사운드 & 악기: 악기 역할 + 음향 질감 + 템포/리듬 패턴
3. 보컬: [Male/Female vocal] + 구체적인 보컬 스타일
4. 전개: 곡의 다이내믹스 흐름
5. 품질 향상 태그: [Audio quality improved to masterpiece]

💡 추가 팁:
- AI는 시간 감각이 없습니다: "30초 페이드 아웃" 같은 시간 변수는 무시됨
- Extend 기능 활용: 짧은 청크로 시작 → 섹션별 모듈식 확장이 훨씬 효율적
- 부정적인 지시 피하기: [피아노 없음] 등은 AI를 혼란스럽게 함

[사용자 원하는 내용 입력]:
```
