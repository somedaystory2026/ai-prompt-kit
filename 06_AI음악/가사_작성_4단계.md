# 가사 작성 4단계 통합 (Suno 구조 태그 포함)

> 💡 K-Pop 메인 작사가 + Suno 전문 작사가 + 프로듀싱 전문가 + 가사 분석 4단계 통합

## 1단계 — 가사 분석

```
내가 좋아하는 "[가사 제목]"이야.
이 가사를 분석해서, 라임배치는 어떻게하는지, 단어구성은 어떤식으로 하는지,
라임의 구성, 자음모음은 어떤식으로 구성하는지 면밀하게 분석부터해서,

[Intro] [Verse] [Pre Chorus] [Chorus] 이런식으로 정리해줘
```

## 2단계 — 제목 10개 생성

```
나는 지금부터 가사를 쓸거야.
장르는 [장르]이고, [내용]에 관한 내용을 쓸 건데,
제목부터 한국어로 10개 써줘
```

## 3단계 — 1절 초안 작성

```
나는 "[선택한 제목]" 이 마음에 들어.
이것에 대한 가사를 Verse / Pre Chorus / Chorus 파트로 나눠서 1절만 써줘
```

## 4단계 — 전체 구조 완성

```
지금까지 쓴걸 토대로 구성을 짜서 가사를 써주는데,
1절부터 이 순서대로 써줘:
Verse1 - PreChorus1 - Chorus1 - Verse2 - PreChorus2 - Chorus2 - Bridge - Chorus3
```

## Suno 가사 7대 규칙

1. 실제 사람이 말하는 것처럼 자연스럽게 작성
2. 클리셰 단어 사용 금지 → `dream, shining, sky, tonight, forever, tapestry` 등
3. 후렴은 짧고 중독성 있게 반복 (틱톡·숏폼 바이럴 기준)
4. 각 Verse는 스토리가 진행되도록 작성
5. 감정선이 Intro → Final Chorus로 갈수록 점층적으로 커지도록 구성
6. Suno가 노래하기 쉬운 음절 길이로 작성 (짧은 문장 위주)
7. 전체 3,000자 이내 유지 (Suno 입력 제한 고려)

**애드립 표기:** 소괄호 사용 → `(yeah) (오늘도) (uh)`
