# 채널 5단계 딥 벤치마킹 분석

> **추천 도구**: NotebookLM

## 프롬프트

```
1단계. 제목 분석
이 채널의 상위 조회수 영상 제목 구조를 분석해줘.
- 구조적 문장형: 질문형/단정형/경고형/나열형 비율
- 강렬한 단어 추출 (충격, 실화, 절대, 1위 등)
- 숫자 활용법: 어떤 숫자가 많이 쓰이고 신뢰/긴박감 주는지
- 클릭률 높았던 제목의 평균 글자 수

2단계. 도입부 분석 (첫 30~60초)
- 첫 문장의 공식: 충격 결과 / 철학적 질문 / 공감 중 어떤 유형?
- 보상 약속: 끝까지 봐야 하는 이유를 어떻게 언급?
- 도입부 속도감: 단문 중심인지 확인

3단계. 본문 분석
- 정보 전달 리듬: 초반에 다 주는지, 야금야금 주는지
- 3~5분 지점 반전 요소나 새로운 갈등이 있는지
- 비유 스타일: 이 채널만의 독특한 메타포
- 권위 인용 방식 (논문·뉴스·사례) 타이밍
- 브릿지 멘트 패턴

4단계. 결론 분석
- 내용 요약 vs 생각할 거리 남기기 중 어떤 방식?
- CTA를 자연스럽게 유도하는 멘트 기법
- 다음 영상 유도 장치 설계 방식

5단계. 말투·어조 딥다이브
- 종결어미 패턴 Top 3
- 단어의 격: 전문 용어 vs 구어체·신조어 비율
- 감정 농도: 냉철한 팩트 전달자 vs 감정 강하게 표현
- 시그니처 문구 추출
- 가상 페르소나 설정 (나이대·직업 태도·자주 쓰는 단어 5개)
```
