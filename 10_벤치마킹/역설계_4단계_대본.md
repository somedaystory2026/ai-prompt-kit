# 역설계 4단계 대본 시스템

> **추천 AI**: Claude

## 1단계 — 역설계

```
넌 세계 최고의 유튜버이자 역설계전문가야.
[여기에 벤치마킹 할 영상 주소복사] 이 영상을 분석하고 싶어.
분석은 후킹, 구조, 영상, 예시, 길이등
이 영상을 벤치마킹 할 좋은 요소들은 전부 알려줘.
```

## 2단계 — 대본 만들기

```
넌 이제 세계 최고의 작가이자 스토리텔러야.
앞서 분석해 준 영상의 핵심 역설계 뼈대를 완벽하게 숙지해 줘.
이제 이 영상이 대중에게 먹혔던 '심리적 트리거'와 '핵심 타겟층'은 유지하되,
소재나 앵글을 살짝 비틀어서 단순 복제가 아닌 완벽한 벤치마킹이 될 수 있게 기획해 줘.

아래의 지침에 따라 대본의 전체 구조와 내용을 짜줘:

1. 초반 30초 후킹 (Hook)
기존 영상이 시청자를 붙잡았던 방식을 똑같이 차용하되,
네가 만든 기획에 맞춰 시청자의 이탈을 막을 강력한 오프닝 멘트와 상황을 작성해 줘.

2. 정보 전달의 구조와 스토리텔링 (Body)
기존 영상의 기승전결 흐름을 그대로 가져와.
각 단계별로 어떤 내용이 들어가야 하는지 목차 형태로 정리하고,
시청자를 끝까지 끌고 갈 스토리텔링의 흐름을 잡아 줘.

3. 새로운 예시와 교차 (Examples)
원본 영상의 예시는 절대 쓰지 마.
네가 잡은 흐름을 뒷받침할 수 있도록
사실을 활용한 완전히 새롭고 지적인 비유와 예시를 각 단계마다 배치해 줘.

이 모든것을 지켜서 아래 형식으로 대본을 만들어줘:
- 추천 제목 3가지
- 위 요소를 지킨 오직 대본 텍스트만 있는 형태의 통 대본
- 기획안 형태는 절대 안됨. 오직 대본만
- [00:00] 같은 타임라인이나 편집방식에 대한 이야기로 다 제외할것
- 구어체로 흥미있고 재미있게 대본을 써줄것
```

→ 자세한 방법은 `05_이미지비주얼/롱폼_졸라맨_이미지.md` 참고
