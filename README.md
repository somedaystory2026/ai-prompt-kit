## 🤖 AI 도구별 역할

| AI | 주요 역할 |
|----|-----------|
| **Gemini** | 영상 분석 → 릴스 분석, 흑백요리사 |
| **Claude** | 구조 분석·긴 출력 → 피드/프로필/카드뉴스/쇼츠 시스템/글쓰기/벤치마킹/풀 파이프라인 |
| **ChatGPT** | 이미지 이해·프롬프트 → 스타일 추출/해부학/지브리/클레이 ASMR |
| **Grok** | 일관성 영상 생성 → 60초 영상/자동화 20개/Google Flow 자동화 |
| **NotebookLM** | 채널 딥 분석 → 5단계 벤치마킹 → 마스터 프롬프트 설계 |
| **Suno** | AI 음악 생성 → 마스터 프롬프트·인트로 치트키·K-Pop·일본 시니어·가사 작성 |
| **Claude Code** | 모션 그래픽·카드뉴스 PNG 자동화·캡컷 에이전트·Remotion |

## 📋 프롬프트 사용 방법

1. 원하는 카테고리 폴더로 이동
2. 해당 `.md` 파일 열기
3. 프롬프트 내 `[입력]`, `[여기에 붙여넣기]` 부분을 내용으로 교체
4. 해당 AI 도구에 붙여넣기
