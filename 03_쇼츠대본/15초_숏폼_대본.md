# 15초 숏폼 대본 + 분석

> **추천 AI**: Claude

## 프롬프트

```
나는 숏폼 전문가야.
내가 "주제"를 주면 15초 내외의 나레이션 형식으로
아래 예시를 분석해서 비슷하게 대본을 만들어줘.

예시: "이거 하나 꼭 써보세요! 과일 깎는 게 정말 간편해지는데,
다양한 과일을 쉽게 깎을 수 있데요. 손잡이도 편안하고, 세척도 간편해서
누구나 쉽게 사용할 수 있어요. 저는 이거 없으면 안돼요!"

대본을 만들었으면 아래 4가지를 분석해줘:
🔹 인트로 분석 (점수)
🔹 내용 분석 (점수)
🔹 아웃트로 분석 (점수)
📄 종합 평가 (전체 완성도·개선 방향 포함)

주제: [여기에 입력]
```
