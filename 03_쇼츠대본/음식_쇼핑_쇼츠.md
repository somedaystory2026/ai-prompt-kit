# 음식 쇼핑 쇼츠 3단계 제작 시스템

> **추천 AI**: Claude

## 1단계 — 소재 찾기

```
나는 지금 바이럴이 될 쇼츠를 만들려고 해.
흥미로운 이야기가 있는 음식 소개 쇼츠를 만들건데,
한국인들이 궁금해할만한 신기한 이야기가 있는 음식들을 여러개를 추천해줘.
브랜드까지 짚어주면 좋겠어.
```

## 2단계 — 아이템 조사

```
한국인들이 좋아하는 [제품명]의 영상을 만들거야.
이 아이템의 흥미로운 이야기, 반전 요소들을 찾아줘.
(이왕이면 아마존에 있는 것)
```

## 3단계 — 대본 작성 (50초 이내)

```
너는 유튜브 쇼츠 인기 채널의 프로 크리에이터야.
주제: [제품명]

구조:
- 3초 후킹 (흥미 유발)
- 배경 설명 (지식 전달)
- 반전 혹은 공감 (감성 포인트)
- 마무리 (질문형으로 댓글 유도)

조건:
- 첫 문장: 바로 궁금해할 만한 의문문이나 반전으로 시작
- 스토리텔링: '창업자의 정신', '어머니의 삼시세끼' 등 개인적 서사 포함
- 마무리: 시청자의 경험을 묻는 질문으로 댓글 유도
- 형식: 호흡 끊기는 지점에 슬래시(/)나 줄바꿈 활용
- 어조: 구어체, 친근한 말투, 핵심 위주 짧은 문장
```
