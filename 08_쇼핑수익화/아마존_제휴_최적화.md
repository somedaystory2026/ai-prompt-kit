# 아마존 제휴 최적화 블로그 HTML 생성기

> **추천 AI**: Claude (HTML 출력)

## 작동 방식

1. 내가 아마존 링크들을 주면 그와 관련된 **리뷰 링크**를 찾아줘.
2. 나한테 물어봐: "제품이미지주소", "아마존 제휴 바로가기 링크"를 주세요 라고.
3. 내가 링크를 줄테니까 그것으로 아래 형식에 맞게 출력해줘.

## 프롬프트

```
구조: 반드시 HTML 형식으로 출력.
목차 → 서론-본론(4개 이상 섹션)-결론-FAQ(5개) → 해시태그 10개
영어로 작성 / 문단 사이사이마다 사이트로 이동할 수 있는 바로가기 버튼 만들어줘

내용:
전문적인 어조 / 영어로 / 공백 제외 최소 3,000자 이상 상세하게.
리뷰는 별 5개짜리로 5개만 추려주고, 리뷰 수도 본문에 어필해줘.
제품 이미지는 max-width: 50% 설정으로 중간중간 바로가기랑 같이 넣어줘.
이미지 클릭 시 아마존 판매 페이지로 이동하도록 설정.

버튼 스타일 (보라 그라데이션):
padding: 20px 55px / border-radius: 50px
background: linear-gradient(135deg, #6c5ce7 0%, #a29bfe 100%)
hover: translateY(-5px) + 브라이트니스 올라감

SEO 최적화:
상단에 메타 디스크립션 삽입 (사용자가 클릭해야 할 이유 명확히 제시)
핵심 키워드를 자연스럽게 문맥에 녹여서 구글봇이 전문 분야 콘텐츠로 인식하게 할 것
```
